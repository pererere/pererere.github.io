The last argument specifies whether or not to clone instances (objects that are from a custom class or are not created by the {\ttfamily Object} constructor. This value may be {\ttfamily true} or the function use for cloning instances.

When an {\ttfamily instance\+Clone} function is provided, it will be invoked to clone objects that are not "{}plain"{} objects (as defined by is\+Plain\+Object{\ttfamily is\+Plain\+Object}). If {\ttfamily instance\+Clone} is not specified, this library will not attempt to clone non-\/plain objects, and will simply copy the object reference.\hypertarget{README.md_autotoc_md4103}{}\doxysubsection{\texorpdfstring{Attribution}{Attribution}}\label{README.md_autotoc_md4103}
