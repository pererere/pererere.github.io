An express-\/style development middleware for use with \href{https://webpack.js.org}{\texttt{ webpack}} bundles and allows for serving of the files emitted from webpack. This should be used for {\bfseries{development only}}.

Some of the benefits of using this middleware include\+:


\begin{DoxyItemize}
\item No files are written to disk, rather it handles files in memory
\item If files changed in watch mode, the middleware delays requests until compiling has completed.
\item Supports hot module reload (HMR).
\end{DoxyItemize}\hypertarget{README.md_autotoc_md21994}{}\doxysubsubsection{\texorpdfstring{Getting Started}{Getting Started}}\label{README.md_autotoc_md21994}
