Obtains the machine\textquotesingle{}s default gateway through {\ttfamily exec} calls to OS routing interfaces.


\begin{DoxyItemize}
\item On Linux and Android, the {\ttfamily ip} command must be available (usually provided by the {\ttfamily iproute2} package).
\item On Windows, {\ttfamily wmic} must be available.
\item On IBM i, the {\ttfamily db2util} command must be available (provided by the {\ttfamily db2util} package).
\item On Unix (and mac\+OS), the {\ttfamily netstat} command must be available.

Returns\+: {\ttfamily result} {\itshape Object}
\begin{DoxyItemize}
\item {\ttfamily gateway}\+: The IP address of the default gateway.
\item {\ttfamily interface}\+: The name of the interface. On Windows, this is the network adapter name.
\end{DoxyItemize}

The {\ttfamily .v\{4,6\}()} methods return a Promise while the {\ttfamily .v\{4,6\}.sync()} variants will return the result synchronously.

The {\ttfamily gateway} property will always be defined on success, while {\ttfamily interface} can be {\ttfamily null} if it cannot be determined.
