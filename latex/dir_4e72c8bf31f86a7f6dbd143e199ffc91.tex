Wrap words to the specified column width.\hypertarget{readme.md_autotoc_md22564}{}\doxysubsubsubsection{\texorpdfstring{string}{string}}\label{readme.md_autotoc_md22564}
Type\+: {\ttfamily string}

String with ANSI escape codes. Like one styled by \href{https://github.com/chalk/chalk}{\texttt{ {\ttfamily chalk}}}. Newline characters will be normalized to {\ttfamily \textbackslash{}n}.\hypertarget{readme.md_autotoc_md22565}{}\doxysubsubsubsection{\texorpdfstring{columns}{columns}}\label{readme.md_autotoc_md22565}

Number of columns to wrap the text to.\hypertarget{readme.md_autotoc_md22566}{}\doxysubsubsubsection{\texorpdfstring{options}{options}}\label{readme.md_autotoc_md22566}
Type\+: {\ttfamily object}\hypertarget{readme.md_autotoc_md22567}{}\doxysubsubsubsubsection{\texorpdfstring{hard}{hard}}\label{readme.md_autotoc_md22567}
Type\+: {\ttfamily boolean}~\newline
 Default\+: {\ttfamily false}

By default the wrap is soft, meaning long words may extend past the column width. Setting this to {\ttfamily true} will make it hard wrap at the column width.\hypertarget{readme.md_autotoc_md22568}{}\doxysubsubsubsubsection{\texorpdfstring{word\+Wrap}{word\+Wrap}}\label{readme.md_autotoc_md22568}

By default, an attempt is made to split words at spaces, ensuring that they don\textquotesingle{}t extend past the configured columns. If word\+Wrap is {\ttfamily false}, each column will instead be completely filled splitting words as necessary.\hypertarget{readme.md_autotoc_md22569}{}\doxysubsubsubsubsection{\texorpdfstring{trim}{trim}}\label{readme.md_autotoc_md22569}
