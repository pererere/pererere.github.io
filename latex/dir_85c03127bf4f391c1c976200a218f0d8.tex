{\bfseries{If you are using Sass}} along with this Post\+CSS plugin, do not use the colon {\ttfamily \+:} in your {\ttfamily @value} definitions. It will cause Sass to crash.

Note also you can {\itshape import} multiple values at once but can only {\itshape define} one value per line.
