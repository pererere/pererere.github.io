\doxysubsection{Detailed Description}
Module for serializing and deserializing object into memory and disk. By default, the {\ttfamily v8} implementations are used, but if not present, it defaults to {\ttfamily JSON} implementation. Both serializers have the advantage of being able to serialize {\ttfamily Map}, {\ttfamily Set}, {\ttfamily undefined}, {\ttfamily NaN}, etc, although the JSON one does it through a replacer/reviver.\hypertarget{README.md_autotoc_md10489}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{README.md_autotoc_md10489}
