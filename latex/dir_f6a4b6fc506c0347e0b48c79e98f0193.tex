Returns a TUF client which can be used to retrieve any target from the Sigstore TUF repository. The local TUF cache will be initialized and the TUF metadata files downloaded from the \href{https://sigstore-tuf-root.storage.googleapis.com/}{\texttt{ remote repository}} as part of the initialization process.
