Undici exports a top-\/level Cache\+Storage instance. You can open a new Cache, or duplicate a Cache with an existing name, by using {\ttfamily Cache\+Storage.\+prototype.\+open}. If you open a Cache with the same name as an already-\/existing Cache, its list of cached Responses will be shared between both instances.
