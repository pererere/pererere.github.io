\doxysubsection{Detailed Description}
A rate limiter for Node.\+js streams.\hypertarget{README.md_autotoc_md19918}{}\doxysubsection{\texorpdfstring{API usage}{API usage}}\label{README.md_autotoc_md19918}
This module exports two classes, {\ttfamily Throttle} and {\ttfamily Throttle\+Group}.

{\ttfamily Throttle} creates a single throttled stream, based on {\ttfamily stream.\+Transform}. It accepts an {\ttfamily opts} parameter with the following keys\+:


\begin{DoxyItemize}
\item {\ttfamily opts.\+rate} is the throttling rate, in bytes per second.
\item {\ttfamily opts.\+chunksize} (optional) is the maximum chunk size into which larger writes are decomposed; the default is {\ttfamily opts.\+rate}/10.
\end{DoxyItemize}

The {\ttfamily opts} object may also contain options to be passed to the {\ttfamily stream.\+Transform} constructor.

For example, the following code throttles stdin to stdout at 10 bytes per second\+: \begin{DoxyVerb}process.stdin.pipe(new Throttle({rate: 10})).pipe(process.stdout)
\end{DoxyVerb}


{\ttfamily Throttle\+Group} allows the creation of a group of streams whose aggregate bandwidth is throttled. The constructor accepts the same {\ttfamily opts} argument as for {\ttfamily Throttle}. Call {\ttfamily throttle} on a {\ttfamily Throttle\+Group} object to create a new throttled stream belonging to the group.

For example, the following code creates two HTTP connections to {\ttfamily www.\+google.\+com\+:80}, and throttles their aggregate (downstream) bandwidth to 10 KB/s\+: \begin{DoxyVerb}var addr = { host: 'www.google.com', port: 80 };
var tg = new ThrottleGroup({rate: 10240});

var conn1 = net.createConnection(addr),
    conn2 = net.createConnection(addr);

var thr1 = conn1.pipe(tg.throttle()),
    thr2 = conn2.pipe(tg.throttle());

// Reads from thr1 and thr2 are throttled to 10 KB/s in aggregate
\end{DoxyVerb}
