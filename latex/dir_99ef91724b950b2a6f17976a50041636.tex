\doxysubsection{Detailed Description}
{\itshape unicode-\/match-\/property-\/ecmascript} matches a given Unicode property or \href{https://github.com/mathiasbynens/unicode-property-aliases-ecmascript}{\texttt{ property alias}} to its canonical property name without applying \href{https://github.com/mathiasbynens/unicode-loose-match}{\texttt{ loose matching}} per the algorithm used for \href{https://github.com/tc39/proposal-regexp-unicode-property-escapes}{\texttt{ Reg\+Exp Unicode property escapes in ECMAScript}}.
