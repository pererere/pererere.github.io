\doxysubsection{Detailed Description}
{\itshape unicode-\/match-\/property-\/value-\/ecmascript} matches a given Unicode property value or \href{https://github.com/mathiasbynens/unicode-property-value-aliases}{\texttt{ property value alias}} to its canonical property value without applying \href{https://github.com/mathiasbynens/unicode-loose-match}{\texttt{ loose matching}}, per the algorithm used for \href{https://github.com/tc39/proposal-regexp-unicode-property-escapes}{\texttt{ Reg\+Exp Unicode property escapes in ECMAScript}}.
