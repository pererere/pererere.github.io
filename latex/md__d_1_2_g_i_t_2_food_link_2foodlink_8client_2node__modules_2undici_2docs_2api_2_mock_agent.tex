This method creates and retrieves Mock\+Pool or Mock\+Client instances which can then be used to intercept HTTP requests. If the number of connections on the mock agent is set to 1, a Mock\+Client instance is returned. Otherwise a Mock\+Pool instance is returned.

For subsequent {\ttfamily Mock\+Agent.\+get} calls on the same origin, the same mock instance will be returned.
