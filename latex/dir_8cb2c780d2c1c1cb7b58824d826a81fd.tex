\begin{DoxyItemize}
\item DOM elements
\item Text nodes
\item Attributes
\item Comment nodes
\item Documents
\item Document\+Fragments
\item Doctypes
\item Node\+Lists / Arrays

For custom serialization logic, a "{}serialize"{} event is dispatched on every {\ttfamily Node} which event listeners can override the default behavior on by setting the {\ttfamily event.\+detail.\+serialize} property to a String or other Node.

The "{}serialize"{} event bubbles, so it could be a good idea to utilize event delegation on a known root node that will be serialized. Check the {\ttfamily event.\+serialize\+Target} property to check which {\ttfamily Node} is currently being serialized.\hypertarget{README.md_autotoc_md5678}{}\doxysubsection{\texorpdfstring{Installation}{Installation}}\label{README.md_autotoc_md5678}
