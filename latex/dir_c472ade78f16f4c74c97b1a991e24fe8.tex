Karma plugin for inline sourcemap support.\hypertarget{readme.md_autotoc_md11356}{}\doxysubsection{\texorpdfstring{Motivation}{Motivation}}\label{readme.md_autotoc_md11356}
When loading Browserify bundles with inline source maps (via \href{https://www.npmjs.com/package/karma-browserify}{\texttt{ {\ttfamily karma-\/browserify}}}), the stack traces in Chrome don\textquotesingle{}t mention the original modules. This plugin uses \href{https://www.npmjs.com/package/source-map-support}{\texttt{ {\ttfamily source-\/map-\/support}}} to improve the situation.\hypertarget{readme.md_autotoc_md11357}{}\doxysubsection{\texorpdfstring{Use}{Use}}\label{readme.md_autotoc_md11357}
Install the plugin with {\ttfamily npm}\+: \begin{DoxyVerb}npm install karma-source-map-support
\end{DoxyVerb}
