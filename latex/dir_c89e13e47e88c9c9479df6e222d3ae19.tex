Allows to set \href{https://postcss.org/api/\#processoptions}{\texttt{ {\ttfamily Post\+CSS options}}} and plugins.

All {\ttfamily Post\+CSS} options are supported. There is the special {\ttfamily config} option for config files. How it works and how it can be configured is described below.

We recommend do not specify {\ttfamily from}, {\ttfamily to} and {\ttfamily map} options, because this can lead to wrong path in source maps. If you need source maps please use the \`{}sourcemap\`{} option.

For large projects, to optimize performance of the loader, it is better to provide {\ttfamily postcss\+Options} in loader config and specify {\ttfamily config\+: false}. This approach removes the need to lookup and load external config files multiple times during compilation.\hypertarget{README.md_autotoc_md16553}{}\doxysubsubsubsubsection{\texorpdfstring{{\ttfamily object}}{{\ttfamily object}}}\label{README.md_autotoc_md16553}
