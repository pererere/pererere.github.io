\hypertarget{readme.md_autotoc_md15616}{}\doxysubsection{\texorpdfstring{API}{API}}\label{readme.md_autotoc_md15616}
Uses the command {\ttfamily open} on mac\+OS, {\ttfamily start} on Windows and {\ttfamily xdg-\/open} on other platforms.\hypertarget{readme.md_autotoc_md15617}{}\doxysubsubsection{\texorpdfstring{opn(target, \mbox{[}options\mbox{]})}{opn(target, \mbox{[}options\mbox{]})}}\label{readme.md_autotoc_md15617}
Returns a promise for the \href{https://nodejs.org/api/child_process.html\#child_process_class_childprocess}{\texttt{ spawned child process}}. You would normally not need to use this for anything, but it can be useful if you\textquotesingle{}d like to attach custom event listeners or perform other operations directly on the spawned process.\hypertarget{readme.md_autotoc_md15618}{}\doxysubsubsubsection{\texorpdfstring{target}{target}}\label{readme.md_autotoc_md15618}
Type\+: {\ttfamily string}

The thing you want to open. Can be a URL, file, or executable.

Opens in the default app for the file type. For example, URLs opens in your default browser.\hypertarget{readme.md_autotoc_md15619}{}\doxysubsubsubsection{\texorpdfstring{options}{options}}\label{readme.md_autotoc_md15619}
Type\+: {\ttfamily Object}\hypertarget{readme.md_autotoc_md15620}{}\doxysubsubsubsubsection{\texorpdfstring{wait}{wait}}\label{readme.md_autotoc_md15620}
Type\+: {\ttfamily boolean}~\newline
 Default\+: {\ttfamily true}

Wait for the opened app to exit before fulfilling the promise. If {\ttfamily false} it\textquotesingle{}s fulfilled immediately when opening the app.

On Windows you have to explicitly specify an app for it to be able to wait.\hypertarget{readme.md_autotoc_md15621}{}\doxysubsubsubsubsection{\texorpdfstring{app}{app}}\label{readme.md_autotoc_md15621}
Type\+: {\ttfamily string} {\ttfamily Array}

Specify the app to open the {\ttfamily target} with, or an array with the app and app arguments.

The app name is platform dependent. Don\textquotesingle{}t hard code it in reusable modules. For example, Chrome is {\ttfamily google chrome} on mac\+OS, {\ttfamily google-\/chrome} on Linux and {\ttfamily chrome} on Windows.\hypertarget{readme.md_autotoc_md15622}{}\doxysubsection{\texorpdfstring{Related}{Related}}\label{readme.md_autotoc_md15622}
