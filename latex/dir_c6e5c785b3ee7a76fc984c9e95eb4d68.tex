\item receiving a fatal signal from outside the process
\end{DoxyItemize}

The return value of the function is a function that will remove the handler.

Note that the function {\itshape only} fires for signals if the signal would cause the process to exit. That is, there are no other listeners, and it is a fatal signal.\hypertarget{README.md_autotoc_md19223}{}\doxysubsection{\texorpdfstring{Options}{Options}}\label{README.md_autotoc_md19223}
