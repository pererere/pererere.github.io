\doxysubsection{Detailed Description}
{\bfseries{encoding}} is a simple wrapper around \href{https://github.com/ashtuchkin/iconv-lite/}{\texttt{ iconv-\/lite}} to convert strings from one encoding to another.

\href{http://travis-ci.org/andris9/Nodemailer}{\texttt{ }} \href{http://badge.fury.io/js/encoding}{\texttt{ }}

Initially {\itshape encoding} was a wrapper around {\itshape node-\/iconv} (main) and {\itshape iconv-\/lite} (fallback) and was used as the encoding layer for Nodemailer/mailparser. Somehow it also ended up as a dependency for a bunch of other project, none of these actually using {\itshape node-\/iconv}. The loading mechanics caused issues for front-\/end projects and Nodemailer/malparser had moved on, so {\itshape node-\/iconv} was removed.\hypertarget{README.md_autotoc_md5894}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{README.md_autotoc_md5894}

Where

\item {\bfseries{to\+Charset}} is the characterset to convert the string
\item {\bfseries{from\+Charset}} ({\itshape optional}, defaults to UTF-\/8) is the source charset
\end{DoxyItemize}

Output of the conversion is always a Buffer object.
