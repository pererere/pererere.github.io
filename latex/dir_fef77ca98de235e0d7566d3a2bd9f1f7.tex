\doxysubsection{Detailed Description}
{\itshape Regenerate} is a Unicode-\/aware regex generator for Java\+Script. It allows you to easily generate ES5-\/compatible regular expressions based on a given set of Unicode symbols or code points. (This is trickier than you might think, because of \href{https://mathiasbynens.be/notes/javascript-unicode}{\texttt{ how Java\+Script deals with astral symbols}}.)\hypertarget{README.md_autotoc_md17362}{}\doxysubsection{\texorpdfstring{Installation}{Installation}}\label{README.md_autotoc_md17362}
