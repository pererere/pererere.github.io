\doxysubsection{Detailed Description}
\begin{quote}
Easily add ANSI colors to your text and symbols in the terminal. A faster drop-\/in replacement for chalk, kleur and turbocolor (without the dependencies and rendering bugs). \end{quote}

\hypertarget{README.md_autotoc_md2409}{}\doxysubsection{\texorpdfstring{Why use this?}{Why use this?}}\label{README.md_autotoc_md2409}
ansi-\/colors is {\itshape the fastest Node.\+js library for terminal styling}. A more performant drop-\/in replacement for chalk, with no dependencies.


\begin{DoxyItemize}
\item {\itshape Blazing fast} -\/ Fastest terminal styling library in node.\+js, 10-\/20x faster than chalk!
\item {\itshape Drop-\/in replacement} for \href{https://github.com/chalk/chalk}{\texttt{ chalk}}.
\item {\itshape No dependencies} (Chalk has 7 dependencies in its tree!)
\item {\itshape Safe} -\/ Does not modify the {\ttfamily String.\+prototype} like \href{https://github.com/Marak/colors.js}{\texttt{ colors}}.
\item Supports nested colors, {\bfseries{and does not have the nested styling bug that is present in \href{https://github.com/jorgebucaran/colorette}{\texttt{ colorette}}, \href{https://github.com/chalk/chalk}{\texttt{ chalk}}, and \href{https://github.com/lukeed/kleur}{\texttt{ kleur}}}}.
\item Supports chained colors.
\item Toggle color support on or off.
