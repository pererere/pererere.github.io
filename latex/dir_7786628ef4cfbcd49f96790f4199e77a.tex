HTTP verbs that Node.\+js core\textquotesingle{}s HTTP parser supports.

This module provides an export that is just like {\ttfamily http.\+METHODS} from Node.\+js core, with the following differences\+:


\begin{DoxyItemize}
\item All method names are lower-\/cased.
\item Contains a fallback list of methods for Node.\+js versions that do not have a {\ttfamily http.\+METHODS} export (0.\+10 and lower).
\item Provides the fallback list when using tools like {\ttfamily browserify} without pulling in the {\ttfamily http} shim module.
