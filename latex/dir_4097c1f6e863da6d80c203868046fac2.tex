\doxysubsection{Detailed Description}
{\itshape unicode-\/property-\/aliases-\/ecmascript} offers Unicode property alias mappings in an easy-\/to-\/consume Java\+Script format. It only contains the Unicode property names that are supported in \href{https://github.com/tc39/proposal-regexp-unicode-property-escapes}{\texttt{ ECMAScript Reg\+Exp property escapes}}.
