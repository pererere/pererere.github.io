\begin{DoxyItemize}
\item {\ttfamily path} The path to start in. Defaults to {\ttfamily process.\+cwd()}
\item {\ttfamily ignore\+Files} Filenames to treat as ignore files. The default is `\mbox{[}'.ignore\textquotesingle{}\mbox{]}{\ttfamily . (This is where you\textquotesingle{}d put}.gitignore{\ttfamily or }.npmignore\`{} or whatever.) If multiple ignore files are in a directory, then rules from each are applied in the order that the files are listed.
\item {\ttfamily include\+Empty} Set to {\ttfamily true} to include empty directories, assuming they are not excluded by any of the ignore rules. If not set, then this follows the standard {\ttfamily git} behavior of not including directories that are empty.

Note\+: this will cause an empty directory to be included if it would contain an included entry, even if it would have otherwise been excluded itself.

For example, given the rules {\ttfamily \texorpdfstring{$\ast$}{*}} (ignore everything) and {\ttfamily !/a/b/c} (re-\/include the entry at {\ttfamily /a/b/c}), the directory {\ttfamily /a/b} will be included if it is empty.
\item {\ttfamily follow} Set to {\ttfamily true} to treat symbolically linked directories as directories, recursing into them. There is no handling for nested symlinks, so {\ttfamily ELOOP} errors can occur in some cases when using this option. Defaults to {\ttfamily false}. 
\end{DoxyItemize}
