The date format library has limited ability to parse strings into dates. It can convert strings created using date format patterns (as above), but if you\textquotesingle{}re looking for anything more sophisticated than that you should probably look for a better library (\href{https://momentjs.com}{\texttt{ momentjs}} does pretty much everything).
