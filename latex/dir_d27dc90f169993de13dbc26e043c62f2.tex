\doxysubsection{Detailed Description}
\begin{quote}
Convert bytes to a human readable string\+: {\ttfamily 1337} → {\ttfamily 1.\+34 kB} \end{quote}


Useful for displaying file sizes for humans.

{\itshape Note that it uses base-\/10 (e.\+g. kilobyte). \href{https://web.archive.org/web/20150324153922/https://pacoup.com/2009/05/26/kb-kb-kib-whats-up-with-that/}{\texttt{ Read about the difference between kilobyte and kibibyte.}}}\hypertarget{readme.md_autotoc_md16808}{}\doxysubsection{\texorpdfstring{Install}{Install}}\label{readme.md_autotoc_md16808}
